Talk to NYC Mortgage Data: Gemini Live API, Multi-Agent AI, Cloud Run

A live AI dashboard for interrogating 446,902 HMDA mortgage applications by voice or text.

HMDA mortgage data is a public goldmine for understanding lending fairness, but it usually sits behind expensive consulting tools and SQL skills most people don't have. This project puts a conversational AI copilot in front of 446,902 NYC mortgage applications and lets anyone interrogate the data in natural language, by typing or by talking.

Black applicants in NYC are denied mortgages at 1.66x the rate of White applicants, and the disparity widens as income rises.

Under the hood is a multi-agent system: a central orchestrator powered by Gemini 2.5 Flash routes each question to a specialist (stats, geography, or policy) and stitches the answers back together. Charts are generated on the fly by the AI itself, not pre-built. Voice mode runs on the Gemini Live API with bidirectional audio streaming over WebSocket, and a pipeline animation bar shows what the system is doing at every step.

Stack: Gemini 2.5 Flash and the Gemini Live API on Vertex AI, FastAPI for the orchestrator and three specialist agents, Chart.js for AI-generated visualizations, and Google Cloud Run hosting all five services independently. The frontend is a single HTML file with no build toolchain.
