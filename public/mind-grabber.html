Mind Grabber

Behavior-Based Action Platformer

Click HERE to play the game on itch. It's browser-based, so very easy to play.

A 2D action platformer built in Godot that explores a modular behavior system where players can steal and equip enemy abilities. The goal was to design a system where combat encounters teach new mechanics and expand player options.

The core mechanic is a hand-grabbing system: players can grab behaviors from defeated enemies and equip them. Behaviors use condition-action pairs that control enemy AI, and players can mix and match them. The experience/leveling system increases behavior capacity, encouraging experimentation.

Ability grabbing prototype (player-green / ai-red):

I implemented the behavior architecture using inheritance patterns, allowing easy addition of new behaviors. The system needed to work for both enemy AI and player abilities.

The game features procedural room generation using BSP algorithms that recursively split rooms, carve corridors between layers, and spawn ladders and doors.

The hand-grabbing mechanic required precise collision detection and state management, tracking when the hand is shooting, returning, or holding a behavior, and ensuring smooth transitions between states.

This project reinforced the value of modular, extensible architecture. Designing the behavior system to be reusable for both enemies and players made the codebase cleaner and easier to iterate on. It also highlighted how player progression can be tied directly to gameplay mechanics rather than just stats.
