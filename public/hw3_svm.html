I chose dst_host_srv_diff_host_rate and dst_bytes as my two features because they offer good variability, allowing the data points to spread out across a wider range on the plot, unlike features that are mostly binary and cluster in a small area. During DoS attacks, the services used may vary, leading to higher values for dst_host_srv_diff_host_rate compared to normal traffic. More importantly, since servers are overwhelmed during these attacks, they can't respond with much data, resulting in dst_bytes values being close to 0. This makes these two features effective for identifying patterns related to DoS attacks.

As you can see the models are a lot worse with only two features than it is with all 41 features (which I guess makes sense, more info leads to better guesses). Of course the results are not terrible, with both kernels reaching 92/93% accuracy. Both Linear and RBF perform well in detecting DoS attacks, with high precision 92% and recall 99%. However, both struggle with non-DoS traffic, especially in terms of recall, where the Linear kernel achieves 0.68 and the RBF kernel only 0.65. This indicates that the models miss a significant portion of non-DoS instances. In conclusion the Linear kernel might be a bit better as it slightly outperforms the RBF kernel on non-DoS traffic.

from matplotlib.colors import ListedColormap
import matplotlib.pyplot as plt
           
def plot_decision_regions(X, y, classifier):
    markers = ('s', 'o')
    colors = ('red', 'blue')
    cmap = ListedColormap(colors[:len(np.unique(y))])

    resolution = 0.01
    x1_min, x1_max = X[:, 0].min() - 1, 2.1
    x2_min, x2_max = X[:, 1].min() - 1, 4
    xx1, xx2 = np.meshgrid(
        np.arange(x1_min, x1_max, resolution),
        np.arange(x2_min, x2_max, resolution))

    Z = classifier.predict(np.array([xx1.ravel(), xx2.ravel()]).T)
    Z = Z.reshape(xx1.shape)

    plt.figure(figsize=(10, 6))
    plt.contourf(xx1, xx2, Z, alpha=0.3, cmap=cmap)
    plt.xlim(xx1.min(), xx1.max())
    plt.ylim(xx2.min(), xx2.max())

    for idx, cl in enumerate(np.unique(y)):
        plt.scatter(
            x=X[y == cl, 0], y=X[y == cl, 1],
            alpha=0.8,
            edgecolor=colors[idx]if cl ==1 else 'black',
            facecolors='none' if cl == 1 else colors[idx],
            marker=markers[idx], label=f'Class {cl}',
            linewidth=1.5,
            s=100 if cl == 0 else 200, 
            zorder=2 if cl == 0 else 1)
    
    plt.xlabel('idx_dst_host_srv_diff_host_rate [standardized]')
    plt.ylabel('idx_dst_bytes [standardized]')
    plt.legend(loc='upper left')
    plt.tight_layout()
