Play the Game

Resource Management Strategy Game

Click HERE to play the game on itch. It's browser-based, so very easy to play.

A card-based life simulation built in Godot that explores decision-making under uncertainty. Players manage four resources across four life stages, making choices from event cards that affect their stats. The point of the game is not to fall below zero for any of your four stats.

The core mechanic is progressive information hiding: as players advance through each stage, card information becomes obscured using shader-based blur effects. Early cards show full information, mid-stage cards blur only the stat effects, and late-stage cards blur both description and effects. This reflects how real-life decisions often have uncertain outcomes.

This ended up being a solo project where I implemented virtually everything. The blur system in particular was interesting, using custom shaders applied to UI elements based on game state. The multi-resource tracking system is an exercise in careful balancing to ensure no single stat could be completely ignored, while the stage progression system filtered cards dynamically based on the current life stage.

The typing animation system for narrative was to add a layer of presentation and a 'slow' reveal of the choice you made. Win/lose condition checking needed to track multiple failure states and provide meaningful feedback about which resource caused failure.

This project highlighted how information design affects gameplay. The progressive blur creates tension and forces players to make decisions with incomplete information.
