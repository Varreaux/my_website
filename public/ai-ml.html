AI / Deep Learning / Machine Learning

Modeling, training, evaluation, and deployment. From prototypes to production.

Perceptron & Adaline Models

Implementation of Perceptron and Adaline classifiers from scratch. Includes visualizations, convergence analysis, and experiments with linearly separable and non-linearly separable Iris dataset classifications.

A* Pathfinding Algorithm

Implemented A* search algorithm for 8-directional pathfinding on a grid with angle-change penalties. Optimizes pathfinding by balancing distance costs and smoothness constraints.

Kropki Sudoku Solver (CSP)

Constraint Satisfaction Problem solver for Kropki Sudoku variants. Uses backtracking with MRV and degree heuristics, forward checking, and constraint propagation to solve puzzles with dot-based neighbor constraints.

8-Puzzle Solver (A*)

A* algorithm implementation for solving the classic 8-puzzle problem. Features Manhattan distance and Linear Conflict heuristics, with graph search optimization for finding optimal solutions to sliding tile puzzles.
