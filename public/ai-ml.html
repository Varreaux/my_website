Artificial Intelligence

AI Career Chatbot

AI chatbot for my portfolio that answers questions about my career using my LinkedIn profile and background information.

AI Research Assistant

AI-powered research assistant that plans searches, performs parallel web research, writes comprehensive reports, and emails results. Uses multiple specialized agents working together.

Red-Teaming AI Agents on OpenClaw

Can an AI agent with access to sensitive data hold the line under attack? A red-team dashboard that fires prompt-injection waves at a live agent on Lightning AI's OpenClaw gateway and scores whether each attempt is blocked or bypassed.

Talk to NYC Mortgage Data: Gemini Live API, Multi-Agent AI, Cloud Run

What if you could ask 446,902 NYC mortgage applications a question, in plain English or out loud, and get a chart back? A conversational AI copilot for HMDA lending data with multi-agent routing, voice via Gemini Live, and a live deployment on Google Cloud Run.

A* Pathfinding Algorithm

Interactive demo of A* search on a grid: draw walls, move the start and end, and turn the angle-change penalty up to watch the path straighten out.

Kropki Sudoku Solver (CSP)

Interactive demo of a Kropki Sudoku solver: pick a puzzle and watch backtracking with MRV and degree heuristics fill the grid in real time, including the pure-Kropki case where every value is derived from dot constraints alone.

8-Puzzle Solver (A*)

Interactive sliding-tile demo: pick a puzzle, choose between Manhattan and linear-conflict heuristics, and watch the optimal solution play out tile by tile. Toggling heuristics on the same scramble shows the node-count gap.

Machine Learning

Perceptron & Adaline Models

Implementation of Perceptron and Adaline classifiers from scratch. Includes visualizations, convergence analysis, and experiments with linearly separable and non-linearly separable Iris dataset classifications.

Probe Attack Detection

Logistic regression classifier on the KDD Cup '99 intrusion dataset where probe attacks make up under 1% of records. Compares L1/L2 regularization, identifies the most predictive network features, and shows how undersampling trades recall for precision.

DoS Detection with SVM Kernels

Compares four SVM kernels (linear, RBF, sigmoid, polynomial) for detecting denial-of-service attacks on KDD Cup '99. Three kernels nail it to ~100% with all 41 features; a 2D feature subset visualizes how the linear and RBF boundaries differ in the overlap region.

Phishing URL Detection with Decision Trees

Can a decision tree flag a phishing URL just from the structure of the link itself? Trained on 36k labeled URLs, then walks through which features the tree leans on at each split, like length, redirects, and punctuation patterns.

Phishing URL Detection with AdaBoost

Same phishing URL problem, but instead of one decision tree there's an ensemble of hundreds of shallow ones voting together. Shows how boosting turns weak stumps into a strong classifier, and where stacking deeper trees starts to hurt instead of help.

Bike Rental Demand with Random Forest Regression

What drives bike rental demand on a given day in Seoul? A regression project on the city's bike-sharing data, comparing a single line through the data against a forest of trees that vote together. Shows where a linear model breaks down and how an ensemble picks up the slack.
